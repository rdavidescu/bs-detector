# Project Drive

## Mission

Reduce misleading, manipulative, and low-quality content by giving people better tools to think before they believe.

This project does not try to control content.  
It tries to improve **how people interpret it**.

\---

## Core Principles

### 1\. No censorship

We do not block, hide, or declare content as true or false. We do not replace the user's judgment.

We surface patterns — weak evidence, emotional manipulation, missing context — and let the user decide. Scores are estimates of credibility risk signals, not truth verdicts. The user always sees the reasoning behind the score.

We analyze patterns, not opinions.

\---

### 2\. User-first thinking

The user owns:

* the decision
* the AI model
* the interpretation

We support judgment � we do not replace it.

\---

### 3\. Transparency over authority

No black-box scores.

Every output must be:

* explainable
* traceable to visible signals
* honest about uncertainty

\---

### 4\. Neutrality by design

We avoid:

* political bias
* ideological framing
* emotional positioning

We focus on:

* evidence quality
* reasoning structure
* manipulation patterns

\---

### 5\. Uncertainty is a feature, not a bug

If something is unclear, incomplete, or ambiguous:
we say it.

Overconfidence is considered a failure.

\---

## What We Are Building

A browser-based credibility assistant that:

* analyzes content using user-owned AI
* highlights weak evidence and manipulation signals
* provides structured, explainable scoring
* adapts to different depth levels (Quick / Standard / Deep)

\---

## What We Are NOT Building

* a truth engine
* a censorship tool
* an AI detector
* a political or ideological filter
* a �one score decides everything� system

* �This is definitely true/false� language
* Hidden logic or unexplained scores
* Feature bloat without clear value
* Emotional or fear-based framing

\---

* No hype
* No drama

When unsure between options:

* choose clarity over complexity
* choose transparency over cleverness
