How to get an OpenRouter key
  1. Go to openrouter.ai/keys
  2. Sign up or log in
  3. Click Create Key
  4. Copy the key (starts with sk-or-) and paste it above

How to get a Gemini key
  1. Go to aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click Create API key
  4. Copy the key (starts with AIza) and paste it above

How to get a Grok key
  1. Go to console.x.ai
  2. Sign up or log in
  3. Create an API key
  4. Copy the key (starts with xai-) and paste it above

Your API keys are stored locally in your browser and are only sent to the respective AI provider during analysis.
